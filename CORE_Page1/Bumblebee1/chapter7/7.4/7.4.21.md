### **7.4.21. mintstatus**

mintstatus寄存器保存每个Privilege Mode下的有效中断的中断level。



​                                                      **<center>表 7-7 minstatus寄存器的控制位</center>**

| **域**       | **位** | **描述**                    |
| ------------ | ------ | --------------------------- |
| **MIL**      | 31:24  | Machine Mode的有效中断level |
| **Reserved** | 23：8  | 未使用的域为常数0           |
| **UIL**      | 7:0    | User Mode的有效中断level    |
